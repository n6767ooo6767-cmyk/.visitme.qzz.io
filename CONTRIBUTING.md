# Getting a Subdomain

## Step 1

Fork this repository.

## Step 2

Create a file in:

```text
domains/
```

Example:

```text
domains/alex.json
```

## Step 3

Add:

```json
{
  "owner": "Alex",
  "target": "alex.github.io"
}
```

## Step 4

Create a Pull Request.

## Step 5

Wait for approval.

After approval your subdomain will become active.
