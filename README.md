# visitme.qzz.io

Free subdomains for personal websites, portfolios, projects and games.

## Examples

- alex.visitme.qzz.io
- portfolio.visitme.qzz.io
- game.visitme.qzz.io
- blog.visitme.qzz.io

## Who can use it?

✅ Personal websites

✅ GitHub Pages

✅ Portfolios

✅ Open-source projects

✅ Games

✅ Educational projects

## Rules

Allowed:

- Personal websites
- Portfolios
- Blogs
- Open-source projects
- Educational content

Not allowed:

- Phishing
- Malware
- Spam
- Illegal content
- Trademark impersonation
- Scam websites

## How to get a subdomain

1. Fork this repository.
2. Create a JSON file inside the `domains` folder.
3. Name the file after the subdomain you want.

Example:

`domains/alex.json`

```json
{
  "owner": "Alex",
  "target": "alex.github.io"
}
```

4. Create a Pull Request.
5. Wait for approval.

## Reserved names

The following names cannot be registered:

- admin
- api
- root
- support
- github
- google
- microsoft
- apple
