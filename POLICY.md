# Acceptable Use Policy

## Allowed

- Personal websites
- Portfolios
- Blogs
- Open-source projects
- Educational projects
- Game projects

## Not Allowed

- Malware
- Phishing
- Spam
- Scam websites
- Illegal content
- Trademark abuse

## Reserved Names

The following names cannot be registered:

```text
admin
root
support
api
mail
ftp
github
google
apple
microsoft
```

Violation of these rules may result in immediate removal.
